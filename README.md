# 📝 To-Do List App

A simple full-stack To-Do List web app built with **HTML, CSS (Bootstrap)**, **Node.js**, **Express**, and **MySQL**.

---

## 🔹 Features

- Add tasks  
- Edit tasks  
- Delete tasks  
- Responsive UI  
- Data stored in MySQL

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, Bootstrap  
- Backend: Node.js, Express  
- Database: MySQL
